## Quick Reference

### Essential Commands
```bash
# Interactive build
gradle releaseBuild

# List tasks
gradle tasks

# Clean build
gradle cleanBuild

# Get help
gradle help --task releaseBuild
```

### Essential Files
| File                  | Purpose                          |
|-----------------------|----------------------------------|
